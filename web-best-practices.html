Suggested best practices

  • should be navigable by keyboard

  • Should be easily read by other automated programs, e.g. Google bots
  • Should be easy to use by mouse. e.g. panels shouldn't close immediately when user accidentally shifts mouse off of it
  • Should be able to work at a variety of viewport sizes
  • should work without javascript
